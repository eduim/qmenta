## Server: Setup and start

Run the following commands:

```
  cd apps/api
  pipenv shell
  make run
  make seed_db
```

## Server: Run tests in the server

Run the following commands

```
  cd apps/api
  pipenv shell
  make run
  make test
```

## Client: Start the Client

Run the following commands

```
   cd apps/client
   pnpm install
   pnpm dev
```
